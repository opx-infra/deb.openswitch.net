deb.openswitch.net

This is the repository for the OpenSwitch OPX Operating System.

To install packages from an official release, use this repository. Change 2.3.1 to the desired release.

deb http://deb.openswitch.net/ 2.3.1 main opx opx-non-free

To install packages from a development stream, use this repository.

deb http://deb.openswitch.net/jessie unstable main opx opx-non-free
deb-src http://deb.openswitch.net/jessie unstable opx

You can substitute unstable with any stream below.

unstable Packages built from HEAD
testing Packages manually promoted by maintainers
stable Packages promoted from testing to be included in releases

After importing our key, you can install OPX packages.

sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys AD5073F1
sudo apt-get update
sudo apt-get install opx-dell-vm
